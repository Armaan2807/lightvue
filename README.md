# Light Vue

The Emerging UI Component library
for Vue 3.x & Vue 2.x

## Contribution Guide

# For LightVue on Vue 2

$ cd ./docs

```bash
# Install dependencies
$ npm install

# Preview docs with hot reload at your localhost
$ npm run dev
```

# For LightVue on Vue 3

$ cd ./docs-v3

```bash
# Install dependencies
$ npm install

# Preview docs with hot reload at your localhost
$ npm run serve
```
